## Theming


### Entry Point

```javascript
theme.button
```

### Object

Example below is taken from the Motor Theme.

```javascript

button: {
    // Takes background color prop, if null takes theme value
    backgroundColor: null,
    // Takes font color prop, if null takes theme value

    // Takes border prop, if null takes theme value
    border: 0,
    // Takes margin prop, if null takes theme value
    margin: null,
    // Takes border radius prop, if null takes theme value
    radius: "15px",

      //Takes hover box shadow prop, if null takes theme value

  },

```
